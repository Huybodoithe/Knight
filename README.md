# 3. Cách chơi
## Các phím bấm
|   Key   | Action                  |
|:-------:| ----------------------- |
|   `A`   | Move left               |
|   `D`   | Move right              |
|   `Left Mouse`   | Attack              |
| `Space` | Jump              |
